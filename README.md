# Behemoth - RSS reader [(download app)](https://drive.google.com/open?id=1ObS9BwE9RHtgTXu3lxIJwkSqJkEqmKfk)
Behemoth is an application designed to read RSS news feeds.

## Technologies:
* MVP
* RecyclerView
* Coroutines
* Retrofit
* SimpleXmlConverter
* Architectural Components (Room, ViewModel)
* Glide
* TimeAgo
